# LeetCode 页面更新日志

这里记录 LeetCode 练习页面和 Chrome 扩展提交流程里的用户可见变化。

## 2026-05-18

### 新增

- 记录通过本页编辑器发起的本地提交和运行样例统计。
- 每道题展示提交次数、Accepted 次数、提交出错次数、样例运行出错次数、
  最近代码量、Accepted 耗时、Accepted 平均耗时，以及每 10 行非空代码耗时。
- 新增默认关闭的沉浸刷题流开关，可用按钮或 `Cmd+Option+F` 开启；开启后
  只有完整提交返回 Accepted 才会自动进入下一题，运行样例通过不会触发。
- “猜你想练”浮层新增推荐流入口，也可用 `Cmd+Option+Shift+F` 启动；它会先
  打开推荐第一题，之后每次完整提交 Accepted 都重新计算推荐并进入下一题。
- 每道题的本地题面面板新增数据范围，优先从力扣题面同步，付费题题面缺失
  时使用已有公开 Doocs reference 对应的题面兜底。
- 新增只存在当前浏览器里的“猜你想练”推荐按钮，在 hover 浮层里按练习调度分
  推荐题目；排序合成到期复习、薄弱程度、题目价值、当前适配和疲劳惩罚，
  并支持没有本地提交记录时的高价值冷启动推荐。
- 页尾新增“猜你想练”打分规则折叠说明，解释五个加权分量、本地稳定度、
  可回忆率、冷启动、失败率平滑和同分兜底排序。
- LeetCode 页面底部新增更新日志入口。
