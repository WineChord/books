# LeetCode 页面更新日志

这里记录 LeetCode 练习页面和 Chrome 扩展提交流程里的用户可见变化。

## 2026-05-20

### 新增

- 支持 `problem=<titleSlug>` 或 `id=<frontendId>` 的直接练题链接。打开后会
  清空本地筛选，跳到对应题目，展开答题编辑器，并保留用户已有的本地草稿。
- 每个题目行预览按钮旁新增弯箭头分享挑战入口，会复制随机挑战邀请文案和当前
  题目的直接练题链接，并在分享按钮附近的临时页面提示里展示本次复制的全文；
  提示会自动消失，也会在下一次页面点击时收起。中文邀请文案改为手写的朋友
  互发题口吻，并避开最近用过的语气。
- 每个已打开答题区新增手动补充用例面板，可以保存、查看、删除和清空随官方
  样例一起运行的测试用例。
- 已展开的本地题面会展示从力扣中国区官方题面 HTML 同步来的图片，以及示例前
  的表结构类表格；这些内容随静态数据发布，不在页面运行时请求资产元数据。

## 2026-05-18

### 新增

- 记录通过本页编辑器发起的本地提交和运行样例统计。
- 每道题展示提交次数、Accepted 次数、提交出错次数、样例运行出错次数、
  最近代码量、Accepted 耗时、Accepted 平均耗时，以及每 10 行非空代码耗时。
- 新增默认关闭的沉浸刷题流开关，可用按钮或 `Cmd+Option+F` 开启；开启后
  只有完整提交返回 Accepted 才会自动进入下一题，运行样例通过不会触发。
- “猜你想练”浮层新增推荐流入口，也可用 `Cmd+Option+Shift+F` 启动；它会先
  打开推荐第一题，之后每次完整提交 Accepted 都重新计算推荐并进入下一题。
- 新增默认开启的“跳过 SQL”和“跳过前端”刷题路由开关；开启后，
  `Option+Up/Down` 上下题、顺序流和推荐流会在自动跳转时跳过数据库 SQL 题，
  以及纯前端 JavaScript/TypeScript 题，但列表、搜索和手动打开仍然保留这些题。
- 每道题新增数据范围，优先从力扣题面同步，付费题题面缺失时使用已有公开
  Doocs reference 对应的题面兜底；展示时作为题面末尾的紧凑约束句，不再
  另起一个独立区块。
- 本地题面和数据范围新增 KaTeX 数学渲染。生成脚本会在页面渲染前修复
  `n 2`、`m x n`、`10 9`、下标变量、Big-O、范围和比较式等破碎纯文本。
- 新增只存在当前浏览器里的“猜你想练”推荐按钮，在 hover 浮层里按练习调度分
  推荐题目；排序合成到期复习、薄弱程度、题目价值、当前适配和疲劳惩罚，
  并支持没有本地提交记录时的高价值冷启动推荐。
- 页尾新增“猜你想练”打分规则折叠说明，解释五个加权分量、本地稳定度、
  可回忆率、冷启动、失败率平滑和同分兜底排序。
- LeetCode 页面底部新增更新日志入口。
